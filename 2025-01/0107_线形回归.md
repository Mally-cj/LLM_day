## 线形回归成立要满足5个前提假设：
1. 自变量和因变量是线形可加的
2. 自变量之间不是多重共线性。
> 多重共线性 = 一个自变量的信息，可以被其他自变量“几乎完整地解释掉”。
> 
> 不能x3=x1+x2
> 可以 x1和x2高度相关，但不完全线形相关。
	​
3. 误差项是相互独立的
4. 误差项的方差是常数
5. 误差项是符合高斯分布的(其实只在进行假设检验时，才会严格要求符合正态分布，在大样本下，由于中心极限定理，这个假设可以放宽。)





## 机器学习/深度学习问题分类

1. 预测一个值（回归）
2. 预测一个类别（分类）
3. 发现结构 / 相似性（聚类 / 表征）
4. 排序 / 决策（排序、推荐、控制）
5. 建模不确定性 / 生成过程（概率 / 生成）


| 维度            | 1️⃣ 回归（预测值）   | 2️⃣ 分类（预测类别）      | 3️⃣ 结构发现 / 表征 | 4️⃣ 排序 / 决策              | 5️⃣ 概率建模 / 生成   |                  |
| ------------- | ------------- | ----------------- | ------------- | ------------------------ | --------------- | ---------------- |
| **核心目标**      | 预测连续数值        | 预测离散标签            | 发现数据内在结构      | 选择 / 排序 / 动作             | 学习数据生成分布        |                  |
| **输出结构**      | 实数 ℝ          | 离散类别 / 概率         | 表征向量 / 簇ID    | 排序 / 动作 / 策略             | 概率分布 / 样本       |                  |
| **是否需要标签**    | 是             | 是                 | 否 / 弱监督       | 是 / 延迟反馈                 | 否 / 可条件         |                  |
| **学习的对象**     | 函数 f(x)       | 决策边界 / P(y\| x)            | 相似性结构                    | 策略 π(a\|x)       | 分布 p(x) 或 p(x\|y) |
| **优化目标**      | 误差最小          | 分类损失最小            | 重构 / 相似度      | 期望收益最大                   | 似然 / ELBO       |                  |
| **典型损失函数**    | MSE / MAE     | CE / Focal        | 重构误差 / 对比损失   | Ranking loss / RL reward | NLL / KL        |                  |
| **预测是否是“点值”** | 是             | 是                 | 否             | 否                        | 否               |                  |
| **是否关注不确定性**  | 较少            | 较少                | 不显式           | 间接                       | 核心关注            |                  |

| **深度学习代表**    | MLP           | CNN / Transformer | AE / CL       | Deep Rank / RL           | VAE / Diffusion |                  |
| **典型应用**      | 房价、销量         | 风控、识别             | 分群、降维         | 推荐、控制                    | 生成、异常           |                  |
